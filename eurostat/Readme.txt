Output file will contain these columns 
1) Country
2) Year
3) ISCOStandard (ISCO08,ISCO88)
4) ValidCodesTotal (How many codes are recognized from csv-codes list - including codes with unavailable values)
5) ExtraCodesTotal (All codes not in csv-codes list
For example - 2 digit codes or 3 digit codes we do not understand)
6) MissingCodesTotal (How many codes from csv-codes list were not mentioned in Eurostat)
7) UnavailableCodesTotal (How many codes from csv-codes list were mentioned but did not have employee number)
8) ExtraCodesList (Comma-separated list of all extra codes)
9) MissingCodesList (Comma-separated list of all missing codes)
10) UnavailabeCodesList (Comma-separated list of all unavailable codes)
